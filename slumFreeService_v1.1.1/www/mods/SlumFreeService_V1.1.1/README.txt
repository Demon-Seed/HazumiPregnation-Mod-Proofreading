@Title: Hazumi's Slum Free Service Mod

Use this mod will add an NPC in slum area.

@Author: Evil Gold
@Version: 1.1.1
@Game Version: 1.6.5

Version 1.1.1 update:
Optimization for a part of coding.

Version 1.1.0 update:
Solve the issue of cannot get back home after end of event, and posture can now change automatically during event.

Version 1.0.1 update:
Fixed the compatibility issue of this mod with namelessDev's "Slum Madam" and "Mad Doctor" mod.

@Install
Put "slumFreeService" folder in "mods" folder of following path: PPLSS/www/mods

@Uninstall
Delete "slumFreeService" folder in "mods" folder of following path: PPLSS/www/mods
